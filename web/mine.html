Как майнить
RuCoin

USB-токен JaCarta с аппаратным Стрибогом + Python-скрипт. Соло или пул — на ваш выбор.

Запуск соло-майнинга

Каждый найденный блок полностью ваш. Награда: — RUC за блок.

$ git clone https://github.com/22Chestii22/RuCoin.git $ cd RuCoin $ python3 rucoin.py --solo --worker ВАШ_НИК

Требования

  • USB-токен JaCarta (ГОСТ/JaCarta-2 ГОСТ) с поддержкой Стрибог
  • Linux / macOS / Windows с установленным OpenSC и pcscd
  • Драйвер JaCarta (libjcPKCS11-2.so) — aladdin-rd.ru
  • Python 3.8+ и зависимости: pip install -r requirements.txt

Установка зависимостей

Linux (Arch / Debian / Ubuntu)

$ sudo pacman -S opensc ccid pcsc-tools libp11 # Arch $ # или $ sudo apt install opensc pcscd libp11-dev # Debian/Ubuntu # Драйвер JaCarta (libjcPKCS11-2.so) — скачайте с aladdin-rd.ru $ echo "module: /usr/lib/libjcPKCS11-2.so" | \ sudo tee /usr/share/p11-kit/modules/jacarta.module

macOS

$ brew install opensc $ # Драйвер JaCarta для macOS — сайт Aladdin

Windows

# Установите OpenSC и драйвер JaCarta # PKCS#11 библиотека обычно в C:\Windows\System32\jcPKCS11.dll

RuCoin Pool

Подключитесь к пулу для стабильных выплат. Минимальный порог выплаты — 1 RUC.

Подключение

$ python3 rucoin.py --pool --worker ВАШ_НИК # Или укажите свой пул: $ python3 rucoin.py --pool --url https://rucoin.vercel.app/api/pool --worker ВАШ_НИК

Протокол: HTTP/JSON (упрощённый Stratum-like)
Эндпоинт: https://rucoin.vercel.app/api/pool
Мин. выплата: 1 RUC
Комиссия пула: 0% (пока тестовый режим)

Как это работает

1
Получаете job — майнер запрашивает у пула текущую высоту, prev_hash и difficulty
2
Майните — ищете nonce, дающий хеш с нужным количеством нулей (Streebog на токене)
3
Отправляете шар — найденный блок отправляется на пул для проверки
4
Получаете награду — пул проверяет, добавляет в цепь, начисывает RUC на ваш адрес
