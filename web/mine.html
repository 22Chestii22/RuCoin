Mining

Майнинг RuCoin

Вставь токен → запусти одну команду → майнится на твой кошелёк.

Быстрый старт

git clone https://github.com/yourname/rucoin
cd rucoin
python3 rucoin.py

Всё. Три команды. Дальше майнер сам подхватит твой токен.

Что делает майнер

01
Подхватывает твой токен

Скрипт находит JaCarta, читает публичный ключ с чипа (тот что от Промсвязьбанка).

02
Вычисляет твой адрес

Хэширует ключ Стрибогом — получается уникальный адрес кошелька. Никаких созданий, импортов и seed-фраз.

03
Начинает майнить

Перебирает nonce, хэширует данные через аппаратный Стрибог на токене, проверяет сложность. Нашёл блок — награда твоя.

RuCoin не про скорость майнинга. Это единственная крипта, которую принципиально нельзя майнить на GPU или ASIC — только на физических токенах.

RuCoin — экспериментальный проект. Не является инвестиционной рекомендацией.
