Как майнить
RuCoin

Вам понадобится USB-токен JaCarta (любая модель с поддержкой Стрибога) и компьютер.

USB

JaCarta ГОСТ / JaCarta-2 ГОСТ

USB-токен с поддержкой аппаратного хеширования Стрибог (ГОСТ Р 34.11-2012).

Vendor ID
24dc:0101
PKCS#11 library
/usr/lib/libjcPKCS11-2.so
Hash speed
~0.1 H/s (один блок ~9 секунд)
Hash algorithm
GOST R 34.11-2012 (256-bit Streebog)
Установка

Настройка окружения

Linux (Arch / Debian / Ubuntu)

$ sudo pacman -S opensc ccid pcsc-tools libp11 # Arch $ # или $ sudo apt install opensc pcscd libp11-dev # Debian # Установите драйвер JaCarta: # Скачайте libjcPKCS11-2.so с сайта производителя # и зарегистрируйте модуль: $ echo "module: /usr/lib/libjcPKCS11-2.so" | \ sudo tee /usr/share/p11-kit/modules/jacarta.module

macOS

$ brew install opensc $ # Драйвер JaCarta для macOS скачайте с сайта Aladdin

Windows

# Установите OpenSC и драйвер JaCarta # PKCS#11 библиотека обычно в C:\Windows\System32\jcPKCS11.dll
Запуск

Майнинг в одну команду

Склонируйте репозиторий и запустите скрипт:

$ git clone https://github.com/22Chestii22/RuCoin.git $ cd RuCoin $ python3 rucoin.py ○ RuCoin Miner v0.1 [+] Found token: JaCarta-2 GOST (slot 0x1ffff) [+] Wallet address: RUC6A2F...B8E1 [+] Mining block #1 ... hash found! [+] Block mined! +0.1 RUC

Каждый найденный блок приносит 0.1 RUC. Сложность майнинга — найти хеш с 4 ведущими нулями.
