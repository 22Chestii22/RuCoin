Подключение

$ python3 rucoin.py --pool --worker ВАШ_НИК # Или для соло-майнинга (по умолчанию): $ python3 rucoin.py --solo --worker ВАШ_НИК

Пул использует упрощённый Stratum-подобный протокол по HTTP/JSON. Адрес пула: https://rucoin.vercel.app/api/pool

Установка

Настройка окружения

Linux (Arch / Debian / Ubuntu)

$ sudo pacman -S opensc ccid pcsc-tools libp11 # Arch $ # или $ sudo apt install opensc pcscd libp11-dev # Debian # Драйвер JaCarta (libjcPKCS11-2.so) — скачайте с aladdin-rd.ru $ echo "module: /usr/lib/libjcPKCS11-2.so" | \ sudo tee /usr/share/p11-kit/modules/jacarta.module

macOS

$ brew install opensc $ # Драйвер JaCarta для macOS — сайт Aladdin

Windows

# Установите OpenSC и драйвер JaCarta # PKCS#11 библиотека обычно в C:\Windows\System32\jcPKCS11.dll
Запуск

Майнинг в одну команду

Клонируйте репозиторий и выберите режим:

$ git clone https://github.com/22Chestii22/RuCoin.git $ cd RuCoin $ python3 rucoin.py --pool --worker my_rig # Пул-майнинг $ python3 rucoin.py --solo --worker my_rig # Соло-майнинг

Флаги:

  • --pool — пул-майнинг (подключается к api/pool)
  • --solo — соло-майнинг (локальная цепочка, полная награда)
  • --worker NAME — имя воркера (для статистики пула)
  • --url URL — кастомный эндпоинт пула (опционально)
